## Project Overview

Built directly from Figma designs, this project focuses on high-fidelity, pixel-perfect and optimized implementation.

## Tech Stack

Framework: Next.js (React)
Styling & Animation: Tailwind CSS
Deployment: Vercel
Key Features
Responsive Layout: Fully optimized for seamless viewing across all device sizes.
Reusable Components: Developed with a modular architecture for scalability.
Image Optimization: Leveraging Next.js for high performance and fast load times.
